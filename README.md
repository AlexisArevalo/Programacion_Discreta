## Descripción

Este proyecto corresponde al Taller 3 de la asignatura Matemáticas Discretas I.

Se implementan algoritmos relacionados con:

- Criptografía clásica
- Criptografía RSA
- Computación Multipartita Segura (MPC)
- Grafos
- Coloreo de grafos
- Álgebra de Boole
- Entropía de Shannon
- Computación cuántica básica

Todos los algoritmos fueron implementados siguiendo los requerimientos del taller.

## Ejecución

Clonar el repositorio

```bash
git clone https://github.com/usuario/Programacion_Discreta.git
```

Entrar al proyecto

```bash
cd Programacion_Discreta
```

Instalar dependencias

```bash
pip install -r requirements.txt
```

Ejecutar

```bash
python src/main.py
```
